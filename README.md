# Restaurant Booking Agent (OpenClaw Skill)

AI-powered restaurant booking and customer service agent built for OpenClaw — handles reservations, menu questions, dietary accommodations, and complaint escalation via Telegram and WhatsApp.

## Also Available For

- [Hermes Agent version](https://github.com/similarcreatures2k13/restaurant-agent-demo)

## What It Does

- Takes reservation requests (date, time, party size, name, phone)
- Answers menu and dietary questions (halal, vegan, gluten-free)
- Handles hours, location, delivery info
- Natural upselling (sharing platters for groups, weekend specials)
- Escalates complaints to human manager

## Setup

1. Install [OpenClaw](https://github.com/openclaw/openclaw)
2. Copy skill: `openclaw skills install ./`
3. Start chatting: `openclaw chat`
